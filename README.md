# PROJETO DJANGO - MÃOS ORIGINÁRIAS
Este é um projeto criado para o curso Asas para o Futuro do IFRN que tem como principal objetivo tornar mais fácil e prático a venda dos artesanatos produzidos pela Comunidade Potiguara Ibirapi de Lagoa Grande, e tambem de divulgar os artesãos que os produzem.

**INTEGRANTES:** RAFAELA, MICAELE, JOYCE MARIA

## PARA RODAR:
1- Copie o link deste repositório no github e no terminal digite:
        git clone link deste repositório

2 - Instale o Django:
        pip install django

3 - Execute as migrações:
        python manage.py migrate

4 - Iniciar o servidor:
        python manage.py runserver

5 - Digite no navegador para acessar a home do site:
        http://127.0.0.1:8000/maos_originarias/
